# Project Brief

## 서비스명
DS_2

## 한 줄 설명
DS_2 디자인 시스템을 Next.js + Tailwind CSS 기반 웹 컴포넌트로 구현한 디자인 시스템 라이브러리

## 타겟 사용자
디자인 시스템이 필요한 누구나

## 핵심 기능
1. 컴포넌트 라이브러리 — DS_2의 UI 컴포넌트를 웹용으로 구현 (Button, Top Appbar, Bottomsheet 등)
2. 디자인 토큰 시스템 — Color, Typography, Spacing, Corner Radius 등 토큰을 CSS 변수 + Tailwind로 제공
3. 컴포넌트 문서/프리뷰 — 각 컴포넌트의 사용법, variant, props를 확인할 수 있는 문서 페이지
4. 다크/라이트 테마 지원 — DS_2 컬러 시스템 기반 테마 전환
5. Figma 연동 가이드 — Figma 디자인 시스템과 웹 컴포넌트 간 매핑 문서
